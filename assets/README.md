# Place your icon.ico file here

## Icon Requirements:
- Format: .ico
- Recommended size: 256x256 pixels
- Name: icon.ico

## How to create:
1. Create a 256x256 PNG with your Aritas Advisors logo
2. Convert to .ico using:
   - https://convertio.co/png-ico/
   - https://www.icoconverter.com/
   - Photoshop / GIMP
3. Save as icon.ico in this folder

## Color Scheme:
- Primary: #1e3a5f (Deep Navy Blue)
- Secondary: #2c5f8d (Medium Blue)  
- Accent: #c9a961 (Gold)

The icon will be used for:
- Application window title bar
- Taskbar
- .exe file icon
- Start menu (if installed)
